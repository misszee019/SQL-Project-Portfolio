# SQL-Project-Portfolio
Project Description

This project is an Airline Loyalty Program Data Analysis using SQL, focused on exploring and analyzing customer behavior within an airline rewards system. The dataset contains approximately 39,000 records and includes information on customers, flight activities, points earned, and points redeemed.

Using SQL, this project investigates patterns in customer engagement, loyalty tier performance, flight frequency, and reward usage to generate meaningful business insights.

The goal is to understand how customers interact with the loyalty program and identify opportunities for improving customer retention, engagement, and reward strategies.

Through structured queries involving joins, aggregations, and grouping functions, this project demonstrates practical data analysis skills applicable to real-world business problems.
